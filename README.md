# 🧠 ConcursoIA - Sistema Inteligente de Estudos

Sistema completo para estudos de concursos públicos com IA integrada.

## 🚀 Funcionalidades

- 📝 Simulados Inteligentes
- ✍️ Correção de Redação com IA  
- 📊 Dashboard de Desempenho
- 🎯 Personalização por Área

## 🛠️ Tecnologias

- Python + Flask
- HTML, CSS, JavaScript  
- SQLite
- Google Gemini API

## 📦 Instalação

```bash
git clone https://github.com/wdps/NOVO-PROJETO.git
cd NOVO-PROJETO
pip install -r requirements.txt
python app.py
```

## 🌐 Deploy

Configure no Railway com variável: `GEMINI_API_KEY`
